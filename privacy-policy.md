# YuMo Privacy Policy

Last updated: 2026-03-28

This privacy policy describes how YuMo handles information when you use the app.

YuMo is a mobile wallet application that helps users send, receive, and request USDC on Base mainnet.

YuMo does not require users to create a hosted account. Wallet access is based on device-stored wallet data, PIN protection, and optional biometrics.

## Information YuMo Uses

YuMo may use the following categories of information to provide the app experience:

- wallet addresses
- public blockchain transaction data
- recovery phrase and wallet data stored locally on the user's device
- PIN and local security settings stored locally on the user's device
- optional biometric preference stored locally on the user's device
- camera access for scanning QR codes

## Information Stored on Device

YuMo stores wallet-related security data locally on the device using platform storage mechanisms, including secure storage where available.

This may include:

- wallet seed or recovery material
- PIN state
- biometric preference
- wallet readiness and onboarding state

YuMo is designed so this information is used locally on the device rather than through a YuMo-hosted user account system.

## Blockchain Data

Transactions made through YuMo are recorded on public blockchains. Public blockchain data can include:

- wallet addresses
- token transfer amounts
- transaction hashes
- timestamps

Blockchain activity is public by nature and may be visible through block explorers and third-party infrastructure providers.

## Camera Access

YuMo requests camera access only to scan QR codes for payment and wallet actions.

YuMo does not use the camera for background recording.

## Third-Party Services

YuMo currently relies on third-party infrastructure providers to support app functionality, including:

- Base RPC infrastructure
- Pimlico for account abstraction, bundling, and paymaster sponsorship
- Expo services for build and app delivery workflows

These providers may process technical request data required to perform blockchain reads, user operations, or build distribution tasks.

## Data Sharing

YuMo does not sell personal data.

YuMo may share the minimum technical data necessary with infrastructure providers in order to:

- read blockchain data
- send sponsored user operations
- build and distribute mobile application binaries

## Data Retention

Most wallet security data used by YuMo is stored locally on the user's device until the app is removed or local data is reset.

Public blockchain records cannot be deleted by YuMo because they are part of the blockchain.

## Your Choices

Users can:

- choose whether to enable biometrics
- use PIN-based unlock instead
- use the in-app "Reset YuMo" option to delete local wallet data, PIN settings, biometrics preference, and cached transaction history from the device
- uninstall the app
- remove wallet data from their device by clearing the app or deleting the app, subject to blockchain records remaining public

## Children

YuMo is not intended for children.
